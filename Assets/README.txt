Game installation process

Find game path
Editor - Persistent Path/Games
Build - Data Path/Games

Add folder for game inside

Add "app" shortcut
Rename to ensure the shortcut file is just called "app"

Add "gamedata" file. The first line is the display name of the game. The second line is its description
This file can be made as a ".txt" file and then renamed to just be "gamedata"

Add any number of ".png" files. filename start does not matter
